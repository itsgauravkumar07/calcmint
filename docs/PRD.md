# CalcMint - Product Requirements Document (PRD)

## Product Vision

CalcMint is an SEO-first financial toolkit that helps users make informed financial decisions using calculators, guides, and educational resources.

The platform focuses on speed, simplicity, accuracy, and trust.

---

# Goals

- Become one of India's largest finance utility websites.
- Rank organically on Google.
- Build topical authority in finance.
- Monetize through AdSense and affiliate marketing.

---

# Target Audience
